# 时间的形状作业

1.安装git bash

2.git clone https://github.com/vertizon/course.git

3.用processon修改时间的形状图

1）10年改成从18~28

2）标记出当前月、大学毕业月

3）修改周图和实际情况相符

4）修改每周的合计时间，并记住

5）修改每月的合计时间，并记住

4.截图，把图片放入wujinchan文件夹，命名为time.png或time.jpg，用以下命令提交到github

```
git add time.*
git commit -m 'time'
git push origin master
```
